Overview

A bellcurve series. If the type option is notspecified, it is inherited from chart.type.For options that apply to multiple series, it is recommended to addthem to the plotOptions.series options structure.To apply to all series of this specific type, apply it to plotOptions.bellcurve.

  intervals

This option allows to define the length of the bell curve. A unit of thelength of the bell curve is standard deviation.
