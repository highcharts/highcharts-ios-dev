Overview

The Data module provides a simplified interface for adding data toa chart from sources like CVS, HTML tables or grid views. See alsothe tutorial article on the Data module](http://www.highcharts.com/docs/working-with-data/data-module).It requires the modules/data.js file to be loaded.Please note that the default way of adding data in Highcharts, withoutthe need of a module, is through the ~!@[series.dataoption.
