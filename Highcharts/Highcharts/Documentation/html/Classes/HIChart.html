  animation

Set the overall animation for all chart updating. Animation can bedisabled throughout the chart by setting it to false here. It canbe overridden for each individual API method as a function parameter.The only animation not affected by this option is the initial seriesanimation, see plotOptions.series.animation.The animation can either be set as a boolean or a configurationobject. If true, it will use the ‘swing’ jQuery easing and aduration of 500 ms. If used as a configuration object, the followingproperties are supported:durationThe duration of the animation in milliseconds.easingA string reference to an easing function set on the Math object.See the easing demo.

@property (nonatomic, readwrite) id animation

Defaults to true.

Try it

  • Updating with no animation
  • With a longer duration
  • With a jQuery UI easing
