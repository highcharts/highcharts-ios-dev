Overview

An object of key-value pairs for SVG attributes. Attributes in Highcharts elements for the most parts correspond to SVG, but some are specific to Highcharts, like zIndex, rotation, rotationOriginX, rotationOriginY, translateX, translateY, scaleX and scaleY. SVG attributes containing a hyphen are not camel-cased, they should be quoted to preserve the hyphen.
